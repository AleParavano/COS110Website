üìö Explanation

Static Members

Static data members belong to the class, not to individual objects. All objects of the class share the same static data. Static member functions can only access static members and don't need an object to be called. Use the scope resolution operator (::) to access them.

Const Objects and Functions

A const object cannot modify its data members. A const member function promises not to modify the object. These constraints help prevent accidental modifications and make code safer.

Initialization Lists

Initialization lists initialize member variables before the constructor body executes. They're especially important for const members, references, and base classes, as these must be initialized before the constructor body runs.

Getter and Setter Functions

Getter functions return the value of private members. Setter functions allow controlled modification of private members. This provides encapsulation while allowing external access in a controlled manner.

Friend Functions

A friend function is not a member of a class but has access to its private members. Friendship is declared inside the class using the friend keyword. It breaks encapsulation, so use carefully.

Create an Employee class that tracks the total number of employees using a static member. Each employee should have an ID, name, and salary.

#include #include using namespace std; class Employee { private: int id; string name; double salary; static int totalEmployees; public: Employee(int i, string n, double s) : id(i), name(n), salary(s) { totalEmployees++; } ~Employee() { totalEmployees--; } static int getTotalEmployees() { return totalEmployees; } void display() const { cout << "ID: " << id << ", Name: " << name << ", Salary: " << salary << endl; } }; int Employee::totalEmployees = 0; int main() { Employee e1(1, "Alice", 50000); Employee e2(2, "Bob", 60000); e1.display(); e2.display(); cout << "Total employees: " << Employee::getTotalEmployees() << endl; return 0; }

Create a 2x2 Matrix class with methods to add, multiply, and display matrices. Use friend functions for matrix operations.

#include using namespace std; class Matrix { private: double data[2][2]; public: Matrix() { for(int i = 0; i < 2; i++) for(int j = 0; j < 2; j++) data[i][j] = 0; } void set(int i, int j, double val) { data[i][j] = val; } friend Matrix add(Matrix a, Matrix b); friend void display(Matrix m); }; Matrix add(Matrix a, Matrix b) { Matrix result; for(int i = 0; i < 2; i++) for(int j = 0; j < 2; j++) result.data[i][j] = a.data[i][j] + b.data[i][j]; return result; } void display(Matrix m) { for(int i = 0; i < 2; i++) { for(int j = 0; j < 2; j++) { cout << m.data[i][j] << " "; } cout << endl; } } int main() { Matrix m1, m2; m1.set(0, 0, 1); m1.set(0, 1, 2); m1.set(1, 0, 3); m1.set(1, 1, 4); m2.set(0, 0, 5); m2.set(0, 1, 6); m2.set(1, 0, 7); m2.set(1, 1, 8); cout << "Matrix 1:" << endl; display(m1); cout << "Matrix 2:" << endl; display(m2); Matrix result = add(m1, m2); cout << "Sum:" << endl; display(result); return 0; }
