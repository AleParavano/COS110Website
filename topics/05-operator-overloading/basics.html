🔀 Operator Overloading - Basics

📚 Explanation

What is Operator Overloading?

Operator overloading allows you to redefine how operators work with custom classes. For example, you can make the + operator work with your own class to add two objects. This makes your custom types behave intuitively like built-in types.

Unary Operators

Unary operators work on a single operand. Examples include ++ (increment), -- (decrement), and - (negation). Unary operators as member functions take no parameters (the object itself is the operand).

Binary Operators

Binary operators work on two operands. Examples include + (addition), - (subtraction), * (multiplication), and == (equality). As member functions, binary operators take one parameter (the right operand).

Member vs Non-Member Overloading

Operators can be overloaded as member functions or non-member functions. Member functions have access to the class's private members but the object is always the left operand. Non-member functions (often friends) offer more flexibility.

Operators That Cannot Be Overloaded

Some operators cannot be overloaded: . (dot), .* (pointer to member), :: (scope resolution), ?: (conditional), and sizeof. These are fundamental to C++ syntax.

⚡ Intermediate Challenge: Complex Number Operators

Create a Complex class and overload +, -, * for complex number arithmetic.

#include using namespace std; class Complex { public: double real, imag; Complex(double r = 0, double i = 0) : real(r), imag(i) {} Complex operator+(const Complex& other) { return Complex(real + other.real, imag + other.imag); } Complex operator-(const Complex& other) { return Complex(real - other.real, imag - other.imag); } Complex operator*(const Complex& other) { return Complex( real*other.real - imag*other.imag, real*other.imag + imag*other.real ); } void display() { cout << real << " + " << imag << "i" << endl; } }; int main() { Complex c1(3, 4); Complex c2(1, 2); Complex sum = c1 + c2; Complex diff = c1 - c2; Complex prod = c1 * c2; cout << "Sum: "; sum.display(); cout << "Diff: "; diff.display(); cout << "Product: "; prod.display(); return 0; }
🔥 Advanced Challenge: Vector Class with Operators

Create a Vector class with +=, -=, and [] operators for element access.

#include using namespace std; class Vector { private: double* data; int size; public: Vector(int s) : size(s) { data = new double[size]; for(int i = 0; i < size; i++) data[i] = 0; } Vector& operator+=(const Vector& other) { for(int i = 0; i < size; i++) data[i] += other.data[i]; return *this; } Vector& operator-=(const Vector& other) { for(int i = 0; i < size; i++) data[i] -= other.data[i]; return *this; } double& operator[](int index) { return data[index]; } void display() { cout << "["; for(int i = 0; i < size; i++) { cout << data[i]; if(i < size-1) cout << ", "; } cout << "]" << endl; } ~Vector() { delete[] data; } }; int main() { Vector v1(3); Vector v2(3); v1[0] = 1; v1[1] = 2; v1[2] = 3; v2[0] = 4; v2[1] = 5; v2[2] = 6; cout << "v1: "; v1.display(); cout << "v2: "; v2.display(); v1 += v2; cout << "After v1 += v2: "; v1.display(); return 0; }
