What is a Pointer?

A pointer is a variable that stores the memory address of another variable. Instead of holding the actual value, a pointer holds the location in memory where that value is stored. This is one of the most powerful features of C++, enabling dynamic memory management, efficient data passing, and the construction of complex data structures like linked lists, trees, and graphs.

Why Use Pointers?

  • Dynamic Memory Allocation: Create variables during runtime using new and delete
  • Efficient Parameter Passing: Pass large objects by reference instead of copying
  • Building Complex Structures: Create linked lists, trees, and graphs
  • Polymorphism: Use base class pointers to access derived class objects
  • Function Pointers: Treat functions as first-class data

The & and * Operators

The & operator (address-of): Gives you the memory address of a variable. If you have an int x, then &x gives the address where x is stored.

The * operator (dereference): Accesses the value at a memory address. If ptr points to x, then *ptr gives you the value of x.

Declaring Pointers

  • int* ptr; - pointer to integer
  • double* ptr; - pointer to double
  • char* ptr; - pointer to character
  • int** pptr; - pointer to a pointer to integer

Key Concepts

Every variable occupies memory with three key properties: an address (where it is), a value (what it contains), and a type (what kind of data it is). Pointers allow us to work with addresses rather than just values, giving us powerful capabilities for dynamic memory management and complex data structures.

‚ö° Intermediate Challenge: Swap Values Using Pointers

Write a function that takes two pointers to integers and swaps the values they point to. The original variables should be modified through the pointers.

#include using namespace std; void swap(int* a, int* b) { int temp = *a; *a = *b; *b = temp; } int main() { int x = 10, y = 20; cout << "Before: x = " << x << ", y = " << y << endl; swap(&x, &y); cout << "After: x = " << x << ", y = " << y << endl; return 0; }

Explanation: This solution uses a temporary variable to hold one value while we overwrite it. We then restore the original value to the second pointer.

üî• Advanced Challenge: Array Manipulation with Pointers

Write functions to: (1) Find the maximum value in an array using pointer arithmetic, (2) Reverse an array in-place using pointers, (3) Sum all elements using pointer traversal.

#include using namespace std; int findMax(int* arr, int size) { int maxVal = *arr; for(int* p = arr; p < arr + size; p++) { if(*p > maxVal) maxVal = *p; } return maxVal; } void reverseArray(int* arr, int size) { int* left = arr; int* right = arr + size - 1; while(left < right) { int temp = *left; *left = *right; *right = temp; left++; right--; } } int sumArray(int* arr, int size) { int sum = 0; for(int* p = arr; p < arr + size; p++) { sum += *p; } return sum; } int main() { int arr[5] = {3, 1, 4, 1, 5}; cout << "Max: " << findMax(arr, 5) << endl; cout << "Sum: " << sumArray(arr, 5) << endl; reverseArray(arr, 5); cout << "Reversed: "; for(int i = 0; i < 5; i++) cout << arr[i] << " "; return 0; }

Explanation: This solution demonstrates pointer arithmetic and manipulation. We iterate through arrays using pointers and perform multiple array operations efficiently.
