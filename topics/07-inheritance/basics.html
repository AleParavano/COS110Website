üå≥ Inheritance - Basics

üìö Explanation

What is Inheritance?

Inheritance allows a derived (child) class to inherit properties and methods from a base (parent) class. This promotes code reuse and establishes relationships between classes. The derived class can add new functionality and override inherited methods.

Base and Derived Classes

The base class defines common functionality. The derived class inherits from the base and adds specialization. Syntax: class Derived : public Base { }. The public keyword specifies the inheritance type.

Access Control in Inheritance

public inheritance: public members stay public, protected stay protected. private inheritance: all members become private. protected inheritance: public becomes protected. Most common is public inheritance.

Constructor and Destructor Chain

When a derived object is created, the base constructor runs first, then the derived constructor. When destroyed, the derived destructor runs first, then the base destructor. This ensures proper initialization and cleanup order.

Method Overriding

Override inherited methods by defining them in the derived class with the same signature. Without virtual, this hides the base method. With virtual, true polymorphic behavior occurs. Use override keyword to indicate intent.

‚ö° Intermediate Challenge: Shape Hierarchy

Create Shape base class with Rectangle and Circle derived classes calculating area and perimeter.

#include #include using namespace std; class Shape { protected: string name; public: Shape(string n) : name(n) {} virtual void display() = 0; virtual double area() = 0; virtual ~Shape() {} }; class Rectangle : public Shape { private: double width, height; public: Rectangle(double w, double h) : Shape("Rectangle"), width(w), height(h) {} void display() override { cout << name << ": " << area() << endl; } double area() override { return width * height; } }; class Circle : public Shape { private: double radius; public: Circle(double r) : Shape("Circle"), radius(r) {} void display() override { cout << name << ": " << area() << endl; } double area() override { return 3.14159 * radius * radius; } }; int main() { Rectangle r(5, 3); Circle c(4); r.display(); c.display(); return 0; }
üî• Advanced Challenge: Employee Hierarchy

Create Employee base with Manager and Developer derived classes with specific bonuses and calculations.

#include #include using namespace std; class Employee { protected: string name; double salary; public: Employee(string n, double s) : name(n), salary(s) {} virtual double calculateBonus() = 0; virtual void display() = 0; virtual ~Employee() {} }; class Manager : public Employee { private: int teamSize; public: Manager(string n, double s, int ts = 0) : Employee(n, s), teamSize(ts) {} double calculateBonus() override { return salary * 0.2; } void display() override { cout << "Manager: " << name << ", Salary: " << salary << ", Bonus: " << calculateBonus() << endl; } }; class Developer : public Employee { private: string language; public: Developer(string n, double s, string lang = "C++") : Employee(n, s), language(lang) {} double calculateBonus() override { return salary * 0.1; } void display() override { cout << "Developer: " << name << ", Salary: " << salary << ", Bonus: " << calculateBonus() << ", Language: " << language << endl; } }; int main() { Manager m("Alice", 50000, 5); Developer d("Bob", 40000, "C++"); m.display(); d.display(); return 0; }
